# assSecretary (Hugo)

**Class:** Utility

**Language:** Java

**Platform:** Windows / Linux

This is our secretary Hugo (Humble Universal Generic Organizer), who helps us organize things.

## Setup

Download our Toolbox-Java (which is a separate project here on github) into an adjacent directory on your hard drive.

Start the build by calling under Windows:

```
build.bat
```

Or under Linux:

```
build.sh
```

## Run

To start up the assSecretary project after it has been built, you can call under Windows:

```
run.bat
```

Or under Linux:

```
run.sh
```
